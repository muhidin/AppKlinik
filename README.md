## Tutorial Menggunakan Aplikasi Klinik
### Clone Repo

```bash
git clone https://github.com/muhidins/AppKlinik.git
cd AppKlinik
composer install
npm install
php artisan migrate --seed
code .
php artisan serve
```
Buka Visual Studio Code dan buka terminal berikan perintah

```bash
npm run dev
```
